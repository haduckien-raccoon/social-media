# Tóm Tắt Chức Năng

## 1. Accounts
- Register, verify email, login/logout
- Reset password
- Profile cá nhân và profile người dùng khác
- Chỉnh sửa email/username/password/profile

## 2. Friends
- Gửi/nhận lời mời kết bạn
- Chấp nhận/từ chối/hủy lời mời
- Danh sách bạn bè + gợi ý

## 3. Posts
- Tạo/sửa/xóa bài viết
- Đăng text, ảnh, file hoặc kết hợp
- Tag bạn bè
- Reaction post/comment (like/love/haha/wow/sad/angry)
- Comment/reply, share post
- Report nội dung

## 4. Groups
- Tạo nhóm, tham gia nhóm
- Duyệt thành viên
- Đăng bài trong nhóm, duyệt bài theo role
- Quản trị nhóm (owner/admin/member)

## 5. Notifications
- Tạo notification theo sự kiện post/friend/group
- SSE realtime theo user channel
- Bell badge + toast realtime global
- Trang notifications: click thông báo để redirect đúng tài nguyên

## 6. Realtime
- WebSocket cho feed và post detail
- SSE cho notifications
